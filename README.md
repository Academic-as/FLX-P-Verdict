# ⚡️ FLX-P Verdict

![FLXP Style](https://img.shields.io/badge/Style-FLXP--Project-8a2be2?style=for-the-badge)
![React](https://img.shields.io/badge/React-20232A?style=for-the-badge&logo=react&logoColor=61DAFB)

Утилита для модераторского состава **FLX-Project**, предназначенная для быстрой и безошибочной генерации команд наказания (ban/warn). 

---

## 🛡 О проекте

Данный инструмент был разработан для оптимизации рутинных действий администрации. Он позволяет свести к минимуму опечатки и ускорить процесс выдачи наказаний, предоставляя удобный графический интерфейс вместо ручного ввода команд в консоль.

> **⚠️ Важно:** Данное приложение **не является обязательным** к использованию. Это вспомогательный инструмент, созданный для личного удобства модераторов. Вы всегда можете продолжать использовать стандартную консоль.

---

## 🔥 Как это упрощает работу?

1.  **Защита от опечаток:** Вам больше не нужно беспокоиться о синтаксисе. Команда собирается автоматически по стандартам сервера.
2.  **Интеллектуальные шаблоны:** Выбирайте тип нарушения из списка, и система сама предложит актуальный срок и корректную формулировку.
3.  **Автоматизация флагов:** Инструмент сам добавит необходимые технические флаги (например, `-g`), основываясь на выбранном типе нарушения.
4.  **Визуальный контроль:** Вы видите итоговую команду перед её использованием, что исключает ошибки в SteamID или времени.
5.  **Единый формат:** Помогает поддерживать чистоту игровых логов, так как все причины пишутся в едином стиле.

---

## 🚀 Функционал

* **Switch Mode:** Быстрое переключение между `BAN` и `WARN`.
* **Smart Time:** Автоматическая подстановка времени для стандартных нарушений.
* **Clipboard Integration:** Копирование команды в один клик.
* **Adaptive UI:** Интерфейс, полностью адаптированный под визуальный стиль FLX-Project.
